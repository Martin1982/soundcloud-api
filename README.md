# SoundCLoud API PHP

This is a PHP wrapper for the [SoundCloud API](https://developers.soundcloud.com/docs/api/reference).

## Requirements
* PHP 7.2 or later.
* Symfony HTTP Client

## Installation
Install it using [Composer](https://getcomposer.org/):

```sh
composer require pouler/soundcloud-api
```
